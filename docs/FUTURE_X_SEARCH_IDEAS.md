## 2. マルチプラットフォーム検索

### 年齢層別プラットフォーム戦略
- 若年層（〜40歳）: Instagram, TikTok優先
- 中年層（40-60歳）: Facebook, Twitter
- 高年層（60歳〜）: Facebook, 公式ウェブサイト

### 統合スコアリング

## 4. 投稿内容の深層分析

### 政治活動パターン認識

### 時系列分析

## 5. 自動確認システム

### 段階的自動化
1. スコア0.9以上：自動承認
2. スコア0.7-0.9：DM/メンションで確認
3. スコア0.5-0.7：人間の確認必須
4. スコア0.5未満：除外

### 検証項目
- 他議員との相互フォロー
- 自治体公式アカウントからのフォロー
- 投稿内容の一貫性
